# js-exercise

Beginner-to-intermediate JavaScript practice projects, built and run as plain Node.js CLI scripts (no HTML/browser UI).

## Projects

| File | Description |
|---|---|
| `greeting_builder.js` | Builds a time-of-day greeting from a first and last name |
| `temperature_converter.js` | Converts between Celsius and Fahrenheit |
| `number_checker.js` | Reports whether a number is positive, negative, zero, even, or odd |
| `string_formatter.js` | Cleans up inconsistent spacing/casing in names |
| `price_calculator.js` | Applies discount then tax to a price |
| `cart_total_calculator.js` | Calculates subtotal, discount, tax, and total for a cart of items |
| `grade_report_generator.js` | Turns a numeric score into a letter grade + feedback |
| `task_list_utilities.js` | Add/complete/remove tasks without mutating the original array |
| `expense_summary.js` | Calculates totals and the largest expense from a list |
| `quiz_app.js` | Interactive multiple-choice quiz with scoring (run in terminal) |
| `weather_app.js` | Fetches live weather for a location (Visual Crossing API, needs your own API key) |
| `github_random_repo_finder.js` | Fetches a random GitHub repo for a chosen language (GitHub Search API) |
